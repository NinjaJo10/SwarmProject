This project is to learn and use flask to
create a web app that will allow users to
register drones and swarms and allow a
simple connectivity and flight.

It uses a basic backend data base and the
Tello drone API as the drones used in the
project are Tello Drones.

Code can be found here:
https://github.com/NinjaJo10/SwarmProject
